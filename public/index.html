Getting Started with KONKMENG

Welcome to KONKMENG! This guide will help you get started with our code analysis platform.

Installation

First, install our CLI tool:

npm install -g konkmen

Basic Usage

Analyze your first piece of code:

konkmen analyze myfile.js

API Key

Get your API key from the dashboard and set it as an environment variable:

export KONKMENG_API_KEY=your_key_here
